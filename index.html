Paraphin Inc.

We build responsive single page apps with Javascript, Node.js, and SASS.

Software Stack - BREW

  • Backbone.js - a minimal framework that makes single page apps easier to develop
  • React.js - a library that uses JSX to make UI components with very fast data binding
  • Express - Node.js's webapp framework, which organizes the backend very well
  • Webpack - packages assets into optimized bundles to reduce network load

Workflow

Some of us use Sublime Text, while others use Atom for text editing. Ben has been working on a cross-platform Sublime plugin to compile SASS files (see github.com/blitzrk/libsass-build-sublime). However, unless you have compilation errors that you're finding hard to debug, webpack handles compiling SASS, so by having a terminal running webpack --watch --progress --colors and another running node server allows you to have a fully functional local deployment. One catch is that you may need to change the myHostname for everyauth.

We like to host on DigitalOcean with a Ubuntu 14.04 droplet. Don't worry about this, though, because we have hooks in place to automate deployment.

Xoshoal

For the backend, we are using a mix of Redis, MongoDB, and Neo4j for data storage. For the frontend, we are using Bourbon and Neat as a SASS framework. For now, we are creating all of our UI elements from scratch.
